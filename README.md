# Security Signal Desk

Security Signal Desk is a security-operations-themed React + TypeScript dashboard project designed to build strong frontend development skills through a realistic, scalable application structure.

The primary goal of this project is not to build a production-ready SOC platform at the beginning. The first priority is to practice React and TypeScript deeply enough to move toward a mid-level frontend skill set.

Over time, the project can grow into a full-stack portfolio application with a backend API, shared contracts, automated tests, Cypress E2E coverage, security-focused workflows, and CI/CD quality gates.
